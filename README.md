# Dashboard de Estoque de Farmacia

Aplicativo web interativo para monitoramento de estoque de medicamentos.

## Funcionalidades

- Alertas automaticos de medicamentos vencidos ou proximos do vencimento
- Identificacao de estoque critico abaixo do minimo
- Graficos interativos de estoque por categoria
- Tabela filtravel com status visual por criticidade
- Download da tabela filtrada em CSV

## Tecnologias

- Python - Streamlit - pandas - Plotly

## Como executar

pip install -r requirements.txt
streamlit run app.py
